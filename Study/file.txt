🔹 파이썬 파일 입출력 기본 문법

1. 파일 열기
f = open("test.txt", "r")   # 읽기 모드
"r" : read (읽기 전용, 기본값)
"w" : write (쓰기 전용, 기존 내용 삭제 후 새로 작성)
"a" : append (추가 모드, 기존 파일 끝에 이어서 작성)
"rb", "wb" : 바이너리 모드 (이미지, 영상 같은 바이너리 데이터 처리할 때)
---
2. 파일에서 읽기
# 파일 전체 읽기
text = f.read()            # 한 줄만 읽기
line = f.readline()        # 모든 줄을 리스트로 읽기
lines = f.readlines()    # for문으로 한 줄씩 읽기
for line in f:
    print(line.strip())
---
3. 파일에 쓰기
f = open("test.txt", "w")
f.write("Hello\n")
f.write("World\n")
f.close()
👉 w 모드는 기존 파일을 덮어씀.
👉 이어서 쓰고 싶으면 "a" 모드 사용.
---
4. 파일 닫기
f.close()
파일을 열었으면 항상 닫아주는 게 안전.
---
5. with 문 (추천)
with open("test.txt", "r") as f:
    for line in f:
        print(line.strip())
with 블록이 끝나면 자동으로 close() 실행돼서 실수 줄여줌.
---
🔹 백준과의 연결
백준에서 쓰는 sys.stdin은 열려 있는 파일 객체 같은 것.
즉,
sys.stdin.readline() → 파일.readline()과 같음
for line in sys.stdin: → for line in f: 와 같음
