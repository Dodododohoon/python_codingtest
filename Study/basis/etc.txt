- try/except    예외처리.
try:
except Error:

  error 종류
IndexError - 리스트, 튜플 등에서 잘못된 인덱스를 참조했을 때 발생   ex) li = [1, 2, 3] print(li[5])
ValueError - 자료형 변환 등에 잘못된 값이 들어왔을 때 발생    ex)int("abc")
TypeError - 자료형이 맞지 않거나 연산 불가능할 때 발생   ex) "3" + 3   # TypeError (str + int 불가능)
ZeroDivisionError - 0으로 나눴을 때 발생  ex) 10 / 0 
KeyError - 딕셔너리에 존재하지 않는 키를 참조했을 때 발생 ex) dic = {"a": 1} print(dic["b"]) 
NameError - 정의되지 않은 변수를 사용할 때 발생 ex) print(x)   # NameError (x가 정의되지 않음)
RecursionError - 재귀가 너무 깊게 들어가서 파이썬이 허용하는 최대 깊이를 초과해서 발생 (최대 1000)

- min, max  최대 최소 : 파이썬은 한방에 구할 수있음.
min(x, y, z, w) -> 제일 작은 수
